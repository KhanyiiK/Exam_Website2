Exploring Asteroid Characteristics with a Bubble Chart

The JavaScript code fetches data from NASA's NEO API, focusing on asteroid characteristics such as velocity, miss distance, and names. Upon selecting a date, the application retrieves the relevant data, processes it, and displays a bubble chart. The chart utilizes the D3.js library to visualize asteroid data in a comprehensive manner.HOVER OVER BUBBLES TO SEE SPECIFIC DATA
